# StudyMind — AI Study Assistant

A personal AI tutor that answers questions from your own notes and PDFs.

## Features

- Upload PDF, DOCX, TXT, MD files
- Ask questions from your own content
- Powered by Google Gemini AI
- No API key needed for users

## Live Demo

👉 [Click here to use the app](https://YOUR-USERNAME.github.io/studymind-frontend)

## Tech Stack

- HTML, CSS, JavaScript
- PDF.js for PDF reading
- Mammoth.js for DOCX reading
- Google Gemini AI (via backend)
